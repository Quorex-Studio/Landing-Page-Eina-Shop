# EINA SHOP — Landing Page

Landing page para **EINA SHOP**, tienda de maquillaje, skincare y accesorios ubicada en Isla de Margarita, Venezuela.

## 🔗 Live
**[einashop.vercel.app](https://einashop.vercel.app)**

## ✦ Stack
- HTML5 + CSS3 vanilla
- JavaScript puro
- Deploy en Vercel

## ✦ Identidad visual
- **Tipografía:** Cormorant Garamond (títulos) · Source Sans 3 (cuerpo)
- **Colores:** `#78293c` · `#e1ae9d` · `#f4e7d7` · `#252024`
- **Estética:** K-beauty, femenina, premium

## ✦ Secciones
- Hero con animación de estrella giratoria
- Marquee animado
- Grid de 5 productos destacados con imágenes reales
- Sección de contacto con formulario → WhatsApp directo
- Footer

## ✦ Integraciones
- WhatsApp Business (`wa.me`) — botones de producto y formulario
- Catálogo de WhatsApp
- Instagram `@einashop_v`

## ✦ Créditos
- Desarrollo por **[Quorex Studio](https://quorex.vercel.app)**
